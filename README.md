# Image to Drawing Converter

Image to Drawing Converter is a web-based application that converts an uploaded image into a pencil-style sketch using basic image processing techniques.

This project demonstrates how images can be processed directly in the browser without using any external libraries or servers.

---

## Features
- Upload any image from local device
- Convert image into sketch/drawing format
- Original image and converted drawing displayed side by side
- Runs completely in the browser
- No installation required

---

## Technologies Used
- HTML
- CSS
- JavaScript
- Canvas API

---

## How It Works
1. User uploads an image.
2. The image is drawn on an HTML canvas.
3. The image is converted to grayscale.
4. Edge detection logic is applied to generate a sketch effect.
5. The processed image is displayed as a drawing.

---

## Use Cases
- Image processing demonstration
- Mini project for college
- Understanding Canvas and pixel manipulation
- Art and design preview
